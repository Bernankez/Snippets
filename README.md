# 练手的

**框架:** vanilla-ts

**目录:**
- `/style.css` 一个合理的字体设置
- `/binary-heap` 最大堆/最小堆
- `/event` event-bus, 事件中心, `emit, on, off`
- `/reactive` vue-reactive
- `/state-machine` js状态机试试
- `/Vuex` 简易的Vuex
- `/clone` 简单的deepClone
- `/is-loop` 判断是否成环
- `/new-operator` 模拟new操作符
- `/promise-like` 一个符合Promise/A+规范的Promise
- `/scheduler` 最多只能同时执行3个任务的调度器
- `/vue-nextTick` vue nextTick原理
- `/vue-use` 模拟Vue.use
